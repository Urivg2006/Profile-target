# Targeta de perfil

## Creeu una aplicació amb Flutter anomenada Pt1_Cognom_Nom que mostri:
* Una AppBar amb el títol "Perfil".

* FloatingActionButton → canvieu el color de fons (backgroundColor).
* Opcional: canvieu el color de fons del body amb Scaffold.backgroundColor.

**English:**

# Profile Card

## Create a Flutter application named Pt1_Surname_Name that displays:
* An AppBar with the title "Profile".
* In the center of the body, a profile image (Image.network or Image.asset).
* Under the image, a Text with the person's name.
* An IconButton (for example, with a message or phone icon).
* A FloatingActionButton on the screen (to add an extra action).

## Add Styles
* Enhance your app by applying visual styles:
* Text → use TextStyle to change font color, size, and weight.
* IconButton → change the icon color.
* FloatingActionButton → change the background color (backgroundColor).
* Optional: Change the background color of the body with Scaffold.backgroundColor.

> **⚠️PD:** The HTML files were originally written in Catalan, but I have included comments with the English translations for better understanding.
